My Hair Routine
===============


**General Info**

Application for saving products, notes and information about hair routine.

Each user has his account to which he logs in.

**Technologies**

*Java 

*Framework JavaFx 

*Relation Database MySQL

*Maven

*Hibernate


**Tools**

*IntelliJ IDEA Community Edition

*JavaFx Scene Builder
